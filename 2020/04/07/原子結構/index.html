原子結構

發表於 2020-04-07 | 分類於 第一章 | 閱讀次數:

化合物組成

化合物由元素組成，元素由原子組成，而原子由質子、中子與電子組成，下圖為一原子

note
電子雲electron cloud
原子核nucleus
質子protons
中子neutrons

原子符號的表示法

週期表的排序是由原子續決定的，由左到右由上到下原子續越來越大
note
Z:原子量
X:元素
A:原子序

同位素

具有相同原子序(質子數)，但中子數不同，造成原子量的不同如
從左到右氫(Hydrogen)、氘(deuterium)、氚(tritium)

價電子(Valence electrons)

價殼為最外層的電子軌道所含的電子數，
八隅體規則(Octet rule):原子或分子傾向滿足價殼有八個電子
價電子怎麼算?
H有1個價電子He有2個價電子(質子數=電子數)
(因為電子軌道填滿了)
Li有1個價電子
Be有2個價電子
B有3個價電子
C有4個價電子…以此類推
Ne有8個價電子
(因為電子軌道填滿了)
Na有1個價電子
Example
1.F 原子序9 價電子=9-2=7
2.N 原子序7 價電子=7-2=5
3.Cl 與F同一族所以是7
4.Si 原子序14 價電子=14-2-8=4
